# Sales Analytics Dashboard

## About this project
I built this project to practice data analysis using a sales dataset. The idea was to find out which regions and product categories perform best, and where the company might be losing money.

## Tools Used
- Python (Pandas) for cleaning and analyzing the data
- Power BI for building the dashboard

## What I found
- Technology products bring in the most profit, Furniture the least
- West region has the highest sales, South the lowest
- A few sub-categories like Tables and Machines are actually running at a loss, even though the overall numbers look fine

## Files
- `analysis.py` - Python code for the analysis
- `SampleSuperstore.csv` - dataset used
- `Sales_Analytics_Dashboard.pbix` - Power BI dashboard file
- `dashboard_screenshot.png` - screenshot of the dashboard

## How to run this
1. Install pandas: `pip install pandas`
2. Run `analysis.py` to see the data analysis
3. Open the `.pbix` file in Power BI Desktop to see the dashboard
